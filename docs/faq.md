# Frequently Asked Questions

## Q: What is Comic Downloader?

A: Comic Downloader is a command-line tool for downloading comics from various online sources.

## Q: How do I install it?

A: You can install it using pip: `pip install comic-downloader`.

## Q: How do I use it?

A: Run the command `main` in your terminal and follow the on-screen instructions. You can use commands like `source`, `s`, `i`, `d`, and `v` to interact with the tool.

## Q: What are the supported comic sources?

A: The tool supports multiple sources. You can see the full list by typing `source` in the tool's command prompt.

## Q: Can I download a specific range of chapters?

A: Yes, you can use the `v` command to download a specific range of chapters. You need to view the comic details first to see the chapter list.

## Q: Where are the downloaded comics saved?

A: The comics are saved in the directory where you run the `main` command. You can also specify a different output directory by running `main <output_path>`.
