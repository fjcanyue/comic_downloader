# Introduction

This tool provides a command-line interface to download comics from various sources.

## Features

- **Multiple Sources**: Switch between different comic sources.
- **Search**: Search for comics by keyword.
- **View Details**: View detailed information about a comic, including chapters.
- **Download**: 
    - Download all chapters of a comic.
    - Download a specific range of chapters.

## Basic Commands

- `source`: Select a comic source.
- `s <keyword>`: Search for a comic.
- `i <index/url>`: View comic details.
- `d <index/url>`: Download a comic.
- `v <options>`: Download a range of chapters.
- `q`: Quit the application.
